  • Create an object list_vectors with elements vec_y1 and vec_y2.
  • Create a for-loop that goes through each element of list_vectors and plots each list of $y$ values against $x$.

Note: The final output should be a 2-dimensional scatter plot with two sets of points, with the $y_1$ and $y_2$ on the vertical axis, and the $x$ values
on the horizontal axis. Remember to always label the axes and add a title to the plot to make it look more professional.

x_values = np.arange(1, 16)

vec_y1 = 3*x_values**3 + 5*x_values**2 - x_values
vec_y2 = x_values**4 - 6*x_values**2 + 2*x_values

list_vectors = [vec_y1, vec_y2]

for vec in list_vectors:
    plt.scatter(x_values, vec)
    plt.xlabel("X values")
    plt.ylabel("Y values")
    plt.title("Plot of y values against x values")

plt.show()

How does the sample size and the number of simulations affect the distribution of the sample means in the code below?

The sample size shapes the distribution of sample means. By increasing the sample size, the distribution becomes narrower and more concentrated around the true population mean. A larger sample size leads to more accurate estimatates of the population mean because outliers and random variations have less of an affect.

Similarly, by increasing the number of simulations, the more accurate estimate of the sampling distribution of the sample mean. A larger number of simulations recudes sampling error therefore a better approximation.

This can be observed in the sample size below as the values along the x-axis are getting smaller thus the data is concentrated along a narrower distribution.
